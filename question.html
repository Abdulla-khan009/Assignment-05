Question-1: What are the different ways to select an element in the DOM?

select by id __getElementById("")
select by class --getElementsByClassName("")
select by tag name --getElementsByTagName("")
select by query selector--querySelector(".Class name")
select by query selectorAll--querySelectorAll(".Class name")

Question-2: What is the difference between innerHTML, innerText, and textContent ?

innerHTML -- reads/writes HTML content (includes tags)
innerText -- gets only visible text (ignores hidden text and extra spaces)
textContent -- gets all text, including hidden elements and extra spaces

Question-3: What is event delegation in the DOM?

Event delegation is a technique where a parent element handles events for its child elements, even if they are added dynamically.

Question-4: What is event bubbling in the DOM?

Event bubbling is a behavior in the DOM where an event starts from the target element and propagates up to its parent elements.
